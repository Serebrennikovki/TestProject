# AI Predictor Service

Проект для отработки основ Python с использованием современных инструментов.  
Реализует сервис предсказаний на основе ИИ-модели (Gemma через Ollama) с системой баланса пользователя, историей операций и интеграцией через FastAPI, PostgreSQL и RabbitMQ.

## Основные возможности

- **Генерация предсказаний** — отправка текстового запроса к модели Gemma (через Ollama) с возвратом ответа.
- **Баланс пользователя** — при регистрации создаётся баланс с нулевым значением.
- **Пополнение баланса** — возможность увеличить баланс на произвольную сумму.
- **Проверка баланса** — перед вызовом предсказания проверяется достаточность средств (1 единица за запрос).
- **История операций** — в личном кабинете отображаются все предсказания, списания и пополнения.
- **Docker‑compose** — полная контейнеризация всех компонентов.

## Стек технологий

| Компонент       | Технология                                |
|----------------|-------------------------------------------|
| Backend        | Python 3.11, FastAPI                      |
| База данных    | PostgreSQL                                |
| Брокер сообщений | RabbitMQ                                 |
| ИИ-модель      | Gemma (через `gemma-server` + Ollama)     |
| Контейнеризация | Docker, Docker Compose                    |

## Требования к окружению

- Docker (версия 20.10+)
- Docker Compose (версия 2.0+)
- (опционально) Python 3.11 для локальной разработки

## Установка и запуск

Проект полностью запускается через Docker Compose. Вручную собирается только основной образ приложения.
сборка основного приложения: в директории /src/app выполнить docker build -t test-project-app .
сборка системы в целом:  в директории /src нужно выполнить 'docker compose up'
установка модели: в контейнере gemma-server выполнить 'ollama run llama3'
